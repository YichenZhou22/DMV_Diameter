"mag_64slice" and "phase_64slice" contain the GRE magnitude and phase images of 20 subjects

"mag_inv_64slice" contains inversed GRE magnitude images

GRE data from "mag_inv_64slice" and "phase_64slice" were used to train the segmentation model for DMVs in this study.

"mask"  contains the corresponding DMV masks of each subject

"mask_demo" contains the mask of a single DMV extracted from subject DMV_03, which is used for the demo.
